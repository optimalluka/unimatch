# UniMatch - Campus Matchmaking Platform

Built for HackPrinceton Fall 2025

Matchmaking platform connecting Princeton students for friendships and relationships through compatibility-based matching.

## Features
- 20-question personality survey with weighted compatibility algorithm
- AI-generated date recommendations using Groq's Llama 3.3 70B
- Gamification system with shipping and leaderboard
- Magic link authentication
- Real-time matching on "Match Monday"

## Tech Stack
- **Frontend:** Next.js, React
- **Backend:** Supabase (database, auth, APIs)
- **AI:** Groq API (Llama 3.3 70B)
- **Auth:** Magic link system (passwordless)

Built in 36 hours at HackPrinceton.
